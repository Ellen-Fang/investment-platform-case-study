# Investment Platform Case Study

A front-end product prototype for an investment-focused community platform, developed from a proposal into a functional website.

## Project Overview

This project originated from a proposal for an investment experience-sharing platform. Although the proposal was not selected, I further developed the concept into a working front-end website to demonstrate the product vision, user flow, and interface design.

The platform is designed to provide users with a space to share investment experiences, browse community content, interact with others, and explore gamified features such as levels and virtual currency.

## My Role

I was responsible for translating the original proposal into a functional website prototype. My work included front-end design and implementation, page structure planning, interface development, and turning product ideas into interactive pages using React, TypeScript, Tailwind CSS, and Vite.

## Project Goals

This project aimed to:
- transform a proposal into a usable product prototype
- explore how investment-related content and community interaction could be presented on a digital platform
- design a clear and engaging user flow
- demonstrate front-end development and interface implementation skills

## Key Features

- Platform introduction and homepage design
- Community content browsing interface
- Interactive product flow and page structure
- Gamified level and virtual currency system
- Responsive front-end layout

## Tech Stack

- React
- TypeScript
- Vite
- Tailwind CSS

## Reflection

This project is mainly a front-end prototype focused on product presentation and interface design. Rather than leaving the concept as a static proposal, I chose to develop it into a working website to better communicate the product idea.

Through this project, I practiced how to turn a concept into a structured interface, plan page flow, and implement a visual prototype with modern front-end tools.

In the future, this project could be extended with features such as user accounts, data management, and back-end integration.
